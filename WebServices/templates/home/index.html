Descripción del conjunto de datos

Los datos que se incluyen y tratan en este sitio web pertenecen al conjunto de datos capturas (correspondiente al año 2018) proporcionado y mantenido por la dirección general de la Policía Nacial de Colombia.

El conjunto de datos Capturas2018 se encuentra disponible en la página web de datos abiertos de Colombia. Este conjunto posee 226.000 registros con 17 diferentes columnas que presentan datos sobre el tipo de délito, el lugar, la edad del capturado, escolaridad y demás.

En la tabla de abajo se pueden ver los registros de las diferentes capturas. Si se desea es posible ordenar los datos o buscar registros, para esto solo es necesario escribir en el cuadro de búsqueda y el sistema mostrará los registros que coincidan en alguna de sus columnas.
